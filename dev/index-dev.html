API + Streamlit (local)

Use esta página apenas em desenvolvimento. O dashboard Streamlit roda localmente e consome a API em http://localhost:8000 (ou na URL que você configurar em API_BASE_URL).

Docs da API (localhost) Abrir Streamlit (localhost)

Como usar

1) Suba a API local: uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

2) Suba o dashboard: API_BASE_URL=http://localhost:8000 streamlit run streamlit_app.py --server.port=8501

3) Abra os links acima (localhost).
